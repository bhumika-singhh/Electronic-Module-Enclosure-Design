# Electronic Module Enclosure Design

**Location:** Dubai, UAE  
**Duration:** Jan–Feb 2025  
**Tools:** SolidWorks, CAD Design, 3D Modeling  

---

## 📌 Overview
This project focuses on designing a robust electronic module enclosure for PCB-based systems. The goal was to create a functional and manufacturable housing solution that ensures protection, efficient space utilization, and proper thermal performance for sensitive electronic components.

---

## 🎯 Objectives
- Design a secure enclosure for PCB and electronic components  
- Ensure proper ventilation for thermal dissipation  
- Optimize internal layout for component placement and accessibility  
- Maintain manufacturability using standard design practices  

---

## 🛠️ Design Features
- Compact and structured enclosure layout  
- Ventilation considerations for heat management  
- Optimized internal space for PCB and connectors  
- Secure housing to protect sensitive electronics  
- Designed with manufacturability in mind  

---

## 🧠 Skills Applied
- 3D CAD modeling using SolidWorks  
- Mechanical design fundamentals  
- Design for Manufacturing (DFM) principles  
- Problem-solving for space and thermal constraints  
- Attention to precision and component alignment
